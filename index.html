JavaScript数组处理函数详解

本视频详细介绍了JavaScript中常用的数组处理函数，包括map、filter、reduce等的实际应用场景和用法。

map() 函数

对数组中的每个元素执行指定操作，并返回新数组

filter() 函数

根据条件筛选数组元素，返回符合条件的新数组

reduce() 函数

将数组元素通过函数累积为单个值

forEach() 函数

对数组的每个元素执行一次提供的函数

map-function.js

// map() 函数示例
// 对数组中的每个元素执行指定操作，返回新数组

const numbers = [1, 2, 3, 4, 5];

// 将每个数字乘以2
const doubled = numbers.map(function(num) {
  return num * 2;
});

// 使用箭头函数简化
const doubledArrow = numbers.map(num => num * 2);

console.log(doubled); // 输出: [2, 4, 6, 8, 10]

// 从对象数组中提取特定属性
const users = [
  { name: '张三', age: 25 },
  { name: '李四', age: 30 },
  { name: '王五', age: 28 }
];

const userNames = users.map(user => user.name);
console.log(userNames); // 输出: ["张三", "李四", "王五"]

// 将数字数组转换为字符串数组
const stringNumbers = numbers.map(num => num.toString());
console.log(stringNumbers); // 输出: ["1", "2", "3", "4", "5"]

函数说明

map() 函数是JavaScript数组的高阶函数，它创建一个新数组，其结果是该数组中的每个元素调用一次提供的函数后的返回值。

参数：回调函数，该函数接收三个参数（当前元素，当前索引，原数组）

返回值：一个新数组，每个元素都是回调函数的结果
